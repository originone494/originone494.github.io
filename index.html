建立博客

我使用了hexo来建立了一个博客，来记录学习过程。

一、安装Hexo并初始化博客

1.安装Hexo

在安装Hexo之前，需要安装Git，Node.js，具体过程可以直接看https://hexo.io/zh-cn/docs/
然后在终端中执行命令
npm install -g hexo-cli

2.初始化博客

  1. 新建一个文件夹作为博客根目录（我的是D:!myths\blog）
  2. 在终端中输入命令进入根目录
    cd D:\!myths\blog
  3. 执行初始化命令
    npx hexo init
  4. 安装依赖包
    npm install
