一、热更新流程

在资源需要变化时，向资源服务器（FTP、HTTP等）上传AB包和资源对比文件。
每次通过游戏客户端进入时：

  1. 下载服务器上的资源对比文件
  2. 用下载下来的资源对比文件和本地文件进行对比，记录需要进行更新的AB包
  3. 将需要更新的AB包进行更新，再更新资源对比文件
  4. 完成后，正式进入游戏

二、准备打包AB包的脚本

在github上下载AssetBundles Browser
可以通过url在包管理器里添加，也可以直接在https://github.com/Unity-Technologies/AssetBundles-Browser直接下载压缩包。
如果是直接在github上下载压缩包，需要将里面的Test文件删除，不然直接在unity里会报错

一、安装Hexo并初始化博客

1.安装Hexo

在安装Hexo之前，需要安装Git，Node.js，具体过程可以直接看https://hexo.io/zh-cn/docs/
然后在终端中执行命令
npm install -g hexo-cli

2.初始化博客

  1. 新建一个文件夹作为博客根目录（我的是D:!myths\blog）
  2. 在终端中输入命令进入根目录
    cd D:\!myths\blog
  3. 执行初始化命令
    npx hexo init
  4. 安装依赖包
    npm install
