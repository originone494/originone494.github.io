如何建立一个博客

一、安装Hexo并初始化博客

1.安装Hexo

在安装Hexo之前，需要安装Git，Node.js，具体过程可以直接看https://hexo.io/zh-cn/docs/
然后在终端中执行命令
npm install -g hexo-cli

2.初始化博客

  1. 新建一个文件夹作为博客根目录（我的是D:!myths\blog）
  2. 在终端中输入命令进入根目录
    cd D:\!myths\blog
  3. 执行初始化命令
    npx hexo init
  4. 安装依赖包
    npm install
    做完上述步骤后，D:!myths\blog文件夹中出现以下文件夹：
_config.yml：Hexo 全局配置文件（核心，用于配置博客标题、主题、部署信息等）； source/：存放博客原始内容，_posts/ 文件夹用于存放所有博客文章（Markdown 格式）； themes/：存放博客主题，默认主题为 landscape； public/：Hexo 生成的静态网页文件（部署时需上传该目录内容）； scaffolds/：文章模板文件，新建文章时会基于模板生成。 ## 3.本地预览博客 1. 生成静态网页文件 `npx hexo generate` 或者 `npx hexo g` 2. 启动本地服务器 `npx hexo server` 或者 `npx hexo s` 3. 打开浏览器，访问http://localhost:4000 ## 4.创作博客文章 4. 新建博客文章 `npx hexo new "blog name"` 使用命令后，文章会生成在source/_posts/目录下 5. 编辑博客文章 用Markdown编辑器打开_posts下的md文件 6. 编辑后执行命令`npx hexo clean && npx hexo generate && npx hexo deploy`，博客里面的内容就会改变

二、在github上配置SSH

  1. 打开git bash
2. 生成秘钥对 `ssh-keygen -t rsa -C "你的github邮箱@xxx.com"` 一路回车即可 1. 查看公钥内容，复制到剪切板上 `cat ~/.ssh/id_rsa.pub` 1. 打开github，新建一个仓库，仓库名为**你的名字.github.io** 2. 在上方导航栏点开仓库的**Setting**，在左侧导航栏找到**Deploy keys** 3. 在**Deploy Keys**界面点击**Add deploy key**按钮 4. **Title**随便填，将之前复制的公钥内容粘贴到**Key**里面，然后点击**Add key** 5. 在git bash中输入命令`ssh -T git@github.com`，如果出现successfully的信息，就代表成功了。

三、部署Hexo博客到Github Pages上

  1. 记录仓库的SSH地址
    如果上述步骤都没问题的话，那么SSH地址应该是git@github.com:你的github的名字/你的github的名字.github.io.git
  2. 安装Hexo部署插件
    在之前建立的博客根目录中在终端中执行命令
    npm install hexo-deployer-git
  3. 编辑部署信息
    打开根目录中的_config.yml文件，找到deploy配置项
要注意冒号后面有个空格 1. 部署博客到github 在终端执行命令： `hexo clean && hexo g && hexo d` 之后访问https://你的用户名.github.io，即可公网访问你的博客 # 参考资料 https://blog.csdn.net/qq_37294163/article/details/103099424 https://blog.csdn.net/weixin_73527660/article/details/156200825
